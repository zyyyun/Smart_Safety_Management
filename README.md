# Smart_Safety_Management
 
사용 언어 : Kotlin, JavaScripts
기술 스택 : Express, Node.js
DB : PostgreSQL

실행방법 : docker-compose up -d로 컨테이너 실행 후
cd server
nodemon server.js

ngrok http 3000

진행사항
1. 시안에 맞춰 진행해서 구현되어있지않은 페이지가 존재.
-관리자->내 정보->CCTV관리->편집->카메라추가 (이후 없음)
-근로자->내 정보->기기관리->기기등록(이후 없음)

그래서 등록과 관련된 API도 추가하지않았습니다. DB에 직접 더미데이터 추가해서 UI가 보이도록만 구성했습니다.

2. DB.md 파일 작성
설계사항을 그냥 시안에 맞춰 작업해서
DB.md에 관련 테이블 주석 및 생성 SQL 문을 추가해두었습니다.

3. 초대,인증,알람 차이점
핸드폰 인증: 핸드폰을 아무렇게나 입력하고, 인증하면 서버에 남는 로그로 인증하도록 코드가 짜여져있습니다.

초대: 메세지를 지금은 자동이 아니라, 작성된 메세지를 전송버튼을 클릭해야 보내지도록 작성되어있습니다.
그리고 초대코드는 그룹을 만들고싶은 사람은 회원가입하자마자 초대코드 화면에 자기코드가 입력되어있으니 그냥 등록하면됩니다. (근로자는 공백)

알람: 핸드폰에 진동이 온다던가 하지않고 일단 DB에 기록이 남고 페이지에 가면 확인할 수 있도록 짜여져있습니다.

4. 위치정보 차이점
일단 GPS 정보를 받아 해당 위치에 근처 카메라가 있으면 해당 위치로 표시되도록 코드를 작성해놨습니다.
