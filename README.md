# Smart_Safety_Management

frontend: Kotlin(Android)
backend: Express, Node.js
database: PostgreSQL

실행방법

-도커 컨테이너 실행
docker-compose up -d

-서버 실행
cd server
npm install
node server.js

-ngrok 실행
ngrok http 3000

3. 기능의 차이점
핸드폰 인증: 핸드폰을 아무렇게나 입력하고, 인증하면 서버에 남는 로그로 인증하도록 코드가 짜여져있습니다.
(EX: 000-0000-0000으로 입력하고 인증버튼을 클릭하면 인증번호 로그가 남음.)
사유 : 실제 번호로 인증 시 유료 서비스를 사용해야 한다는 것을 확인하고 로직만 추가했습니다.

초대: 메세지를 지금은 자동이 아니라, 작성된 메세지를 전송버튼을 클릭해야 보내지도록 작성되어있습니다.

그리고 초대코드는 그룹을 만들고싶은 사람은 회원가입하자마자 초대코드 화면에 자기코드가 입력되어있으니 그냥 등록하면됩니다. (근로자는 공백)

알람: 핸드폰에 진동이 온다던가 하지않고 일단 DB에 기록이 남고 페이지에 가면 확인할 수 있도록 짜여져있습니다.

4. 테스트 스크립트 사용

현재 AI 관련 혹은 기기들과 관련된 로직이 없어서. 스크립트를 이용해 몇몇 기능 확인 가능합니다.

1) 특정 유저 워치 온도 변경 : 이를 통해 위치정보에서 고열을 확인 가능
2) 이벤트 생성 : AI이벤트감지 탭에서 확인 가능 및 알림발생.
